## About the App
EverAfter is an Android application designed to help users track and manage special events.  
The app allows users to create, view, and organize personal events in a simple and structured way.

## Overview
This project was developed as the final assignment in an Android development course.  
The focus of the project was on building a complete Android application, including multiple screens, user interaction, and data handling.

## Main Features
- Creating and managing special events
- Viewing event details
- Navigating between different screens in the app
- User-friendly interface for event tracking

## Technical Details
- Language: Java
- Platform: Android
- UI: XML layouts
- Architecture: Activities and Intents
- Data handling: Local app data

## How to Run
1. Download or clone the repository from GitHub.
2. Open Android Studio.
3. Select "Open an existing project".
4. Navigate to the project folder and open it.
5. Wait for Gradle to sync and build the project.
6. Run the app on an Android emulator or a physical Android device.
